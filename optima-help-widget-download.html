How to Use This File

This page contains a working preview of the Help Widget. To add it to a Canvas lesson:

  1. Copy everything between the <!-- CANVAS CODE START --> and <!-- CANVAS CODE END --> comments below.
  2. In Canvas, edit your lesson page and switch to the HTML editor.
  3. Paste the code at the bottom of your lesson content.
  4. Update the four data- attributes on the main div to match the specific lesson.
  5. Update INBOX_URL and DISCUSSION_URL inside the script.
  6. To change the accent color for a different subject, find-and-replace #67308F and #f0e6f6.

Subject accent colors: Math #67308F · Science #76C043 · ELA #0E5568 · Social Studies #C7922C · Languages #A53E97 · PE #F78F1E · Electives #55C8E8 · Esports #0E1C42

Implementation Reference

What to copy: Everything between the CANVAS CODE START and CANVAS CODE END comments above. This includes the widget div, all CSS, and all JavaScript.

Per-lesson setup: Change the four data- attributes: data-course, data-module, data-lesson, and data-teacher.

URLs to update: Set INBOX_URL to your Canvas conversations compose link. Set DISC_URL to your course's help discussion page.

Subject colors: Find-and-replace #67308F (accent) and #f0e6f6 (light) with your subject's colors from the Optima brand guide.

If Canvas blocks scripts: Move the JavaScript into your Canvas theme's custom JS file (requires admin), or replace with an embedded Microsoft Form using pre-filled URL parameters for course, module, and lesson fields.
